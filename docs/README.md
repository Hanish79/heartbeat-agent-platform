## Documentation Rules

- All material changes must be reviewed.
- The repository version is authoritative.
- Approved business outputs may be copied to Google Drive.
- Every document must have a clear owner and status.
- Generated content is not approved content until a human explicitly approves it.
- Architecture, security, and approval policies cannot be modified by an agent without human authorization.
